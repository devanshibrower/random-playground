# Random Playground

A personal playground for learning React and building interactive UI components.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Running locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Components

Experiments live in `src/app/components/` — each folder is a route you can visit directly:

- `/components/button` — Submit button with loading/success states
- `/components/toggle-with-motion` — Toggle interaction
- `/components/toggle-with-motion-inputtag` — Input tag toggle
